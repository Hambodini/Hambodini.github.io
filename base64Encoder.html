Base 64 Encoder

This is a base 64 encoder. base 64 encoding a method to convert an image into a very long string. it makes the image 33% smaller them before and the image can be decoded back into a image using js and can be stored into a database. one of the downsides of using base 64 is that it is considered very easy to decode and is not secure for confidential information. i think you can make it more secure by using java hash/salt for passwords and such.
